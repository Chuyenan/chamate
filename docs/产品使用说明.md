# Chamate - 本地AI对话平台

## 产品简介

Chamate 是一个支持多AI提供商的本地对话平台，提供智能对话和辩论模式，所有API Key安全存储在本地。

**核心特性：**
- 多AI提供商支持（OpenAI、Anthropic Claude、通义千问、智谱AI）
- 智能对话与辩论模式
- API Key本地加密存储
- 完整的对话历史管理

## 快速开始

### 方式一：Docker部署（推荐）

1. **环境准备**
   - 安装 [Docker](https://www.docker.com/get-started)
   - 安装 Docker Compose

2. **启动服务**
   ```bash
   git clone <repository-url>
   cd chamate
   docker-compose up -d
   ```

3. **访问应用**
   - 前端界面：http://localhost:3000
   - API服务：http://localhost:8000

### 方式二：本地开发

**后端（Python 3.12+）**
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

**前端（Node.js 20+）**
```bash
cd frontend
npm install
npm run dev
```

## 功能使用指南

### API Key 配置

1. 点击左侧菜单「设置」进入配置页面
2. 选择AI提供商标签页
3. 输入对应的API Key
4. 点击「测试连接」验证
5. 保存配置

**API Key获取：**
| 提供商 | 获取地址 |
|--------|----------|
| OpenAI | https://platform.openai.com/api-keys |
| Anthropic | https://console.anthropic.com/ |
| 通义千问 | https://dashscope.console.aliyun.com/ |
| 智谱AI | https://open.bigmodel.cn/ |

### 智能对话

1. 点击「新建对话」创建会话
2. 在下拉菜单选择AI模型
3. 输入消息发送
4. 左侧列表查看历史对话

### 辩论模式

1. 进入「辩论」页面
2. 输入辩论主题
3. 设定您的立场观点
4. 选择参与辩论的AI模型
5. 点击开始，AI将与您进行多轮辩论
6. 辩论结束后查看总结分析

## 常见问题

### API Key配置问题
- **问题**：测试连接失败
- **解决**：检查Key是否正确，确认账户余额充足

### 连接超时
- **问题**：请求超时无响应
- **解决**：检查网络连接，部分API可能需要代理

### 数据存储位置
- **对话历史**：后端 SQLite 数据库（`data/chamate.db`）
- **API Key**：浏览器 IndexedDB（本地加密存储）

### Docker相关
- **问题**：容器启动失败
- **解决**：检查端口 3000/8000 是否被占用
